Fighter Plane
-------------

1. ArrowUP for forward movement
2. ArrowLeft for left movement
3. ArrowRight for right movement
4. "C" for camera change
5. "Q, E" to rotate
6. "Esc" to quit

///Destroy canon bases 
///Stay safe from volcanos


USAGE 
------
1. create a directory "build"
2. open it.
3. use "cmake .."
4. use "make all"
5. then execute :)
